Rust Auth Service

This is a demonstration of a Rust + Axum web service with JWT authentication, admin controls, and license management.

API Endpoints

Authentication

POST /api/v1/register - Register a new user

{ "email": "user@example.com", "password": "secure_password" }

POST /api/v1/login - Login and get JWT token

{ "email": "user@example.com", "password": "secure_password" }

License

GET /api/v1/license - Get user license information

POST /api/v1/license/invalidate-cache - Invalidate license cache

Admin

GET /api/v1/admin/check_admin_status - Check if user has admin privileges

User

GET /api/v1/user/get_calc_details - Get calculation details (requires valid license)
